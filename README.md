## Goal
The goal of Prizm is to provide a modern, efficient, and user-friendly platform for managing personal and professional projects. It aims to **streamline workflows**, **enhance** collaboration, and improve **productivity** through intuitive design and powerful features.

## Architecture
Monolithic architecture built with scalability and maintainability in mind. 
The system is designed to handle a growing number of users and projects while ensuring optimal performance.

### Front-End
The front-end is developed using Vue.js as SPA (Single Page Application) model to provide a seamless and responsive user experience.

### Back-End
The back-end is built around hexagonala architecture using Java with the Spring Boot framework with reactive programming principles to ensure high performance and responsiveness.
